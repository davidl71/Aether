# Web App Archive

The React/Vite web app is retired as an active runtime surface as of March 11, 2026.

Current supported frontends:
- Python/Textual TUI
- Native CLI

This directory remains in the repository only as:
- archived implementation reference
- UI research material
- possible input for a future unified client direction

Do not treat the contents of `web/` as current runtime guidance.

See:
- [WEB_RUNTIME_RETIREMENT.md](../docs/archive/WEB_RUNTIME_RETIREMENT.md)
- [README.md](../README.md)

## What Remains Here

- historical React components and hooks
- archived browser-oriented docs and scripts
- sample UI assets and experiments

## What Is No Longer Supported

- running the Vite dev server as part of the supported local stack
- managing `web` through the active supervisor or service-manager flows
- treating the browser client as the primary consumer of shared platform APIs

Any future browser or unified UI work should be treated as a fresh implementation decision, not a continuation of this retired runtime.
