# Exarp: List Tools

List available exarp-go tools (lint, testing, security, task_workflow, report, etc.).

## Usage

From project root:

```bash
./scripts/run_exarp_go_tool.sh --list
```

Or with Just: `just exarp-list`

In Cursor chat you can ask: "List exarp-go tools" or use the exarp-go MCP tool catalog.
