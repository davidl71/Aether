# Attributions

Aether's current third-party dependency and license summary is maintained in:

- [docs/THIRD_PARTY_LIBRARIES_AND_LICENSES.md](docs/THIRD_PARTY_LIBRARIES_AND_LICENSES.md)

Scope of that document:

- direct Rust dependencies declared by the active `agents/backend/` workspace
- direct build/test crates that are explicitly declared
- external vendor/runtime components referenced by the active platform docs

It does not attempt to be a full transitive dependency manifest.
