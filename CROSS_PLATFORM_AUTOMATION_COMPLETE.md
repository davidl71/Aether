## 🔄 Workflow

### First-Time Setup (Any Platform)

1. **Run setup script**:
   ```bash
   ./scripts/setup_platform_settings.sh --cmake-configure
   ```

2. **Reload VS Code**:
   - `Cmd+Shift+P` → "Developer: Reload Window"

3. **Verify**:
   - IntelliSense works
   - Build succeeds

### Switching Platforms

1. **On new platform**, run setup script again
2. **Script detects** new platform automatically
3. **Regenerates** `settings.json.user` for new platform

## 🛠️ Script Features

### Detection Capabilities

**Platform**:
- macOS: `uname -s` + `uname -m`
- Linux: `uname -s` + `uname -m`
- Windows: `$env:PROCESSOR_ARCHITECTURE`

**Compiler**:
- macOS: `/opt/homebrew/bin/clang++`, `/usr/local/bin/clang++`, `/usr/bin/clang++`
- Linux: `g++` in PATH
- Windows: Visual Studio 2022, MinGW-w64

**Include Paths**:
- macOS: `/opt/homebrew/include`, `/usr/local/include`
- Linux: `/usr/include`, `/usr/local/include`, GCC C++ headers
- Windows: Visual Studio MSVC paths, MinGW paths

**Python**:
- All platforms: Common installation locations + PATH
