# Desktop Agent

Responsibilities:
- Build the desktop client (SwiftUI macOS or Electron) using shared APIs.
- Deliver dashboards, candlesticks, and controls similar to web/iPad.

Quick start:
1. `bash agents/desktop/scripts/setup.sh` (checks tooling).
2. `bash agents/desktop/scripts/run-tests.sh` to build/test the desktop app.
3. Configure Cursor with `cursor-agent.json`.
