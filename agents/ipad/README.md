# iPad Agent

Responsibilities:
- Build the SwiftUI iPad application consuming shared REST/WebSocket APIs.
- Deliver dashboards, candlestick charts, and control surface aligned with the web/TUI.

Quick start:
1. `bash agents/ipad/scripts/setup.sh` (checks Xcode tooling).
2. `bash agents/ipad/scripts/run-tests.sh` to run Swift package/unit tests.
3. Configure Cursor using `cursor-agent.json`.
