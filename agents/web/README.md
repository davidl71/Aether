# Web Agent

Responsibilities:
- Build the React/TypeScript SPA mirroring the active Python/Textual TUI where parity is intended.
- Integrate with REST/WebSocket APIs for dashboards, orders, controls.
- Deliver candlestick charts, maker/taker analytics, and control widgets.

Quick start:
1. `bash agents/web/scripts/setup.sh` (installs Node modules).
2. `bash agents/web/scripts/run-tests.sh` to run lint/tests.
3. Configure a Cursor agent via the provided `cursor-agent.json`.
