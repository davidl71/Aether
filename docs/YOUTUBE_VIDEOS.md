## Adding YouTube Videos to NotebookLM

### Step 1: Create a NotebookLM Notebook

1. Go to [notebooklm.google.com](https://notebooklm.google.com)
2. Click **"+ New"** to create a new notebook
3. Name it: **"TWS API Videos"** or **"Trading Tutorials"**

### Step 2: Add YouTube Video

1. Click **"+ Add source"**
2. Select **"YouTube"** or **"Website"**
3. Paste the YouTube video URL
4. Click **"Add"** and wait for processing

### Step 3: Share the Notebook

1. Click **"⚙️ Share"** (top right)
2. Select **"Anyone with link"**
3. Click **"Copy link"**
4. Save the link

### Step 4: Add to Library

Return to Cursor and say:

```
"Add [notebook-link] to library tagged 'youtube, tws-api, tutorial'"
```

### Step 5: Summarize the Video

Once added to the library, you can summarize the video:

```
"Research this YouTube video about TWS API in NotebookLM and create a summary in docs/video-summaries/tws-api-essential-components.md"
```

## Video Summarization Workflow

### Using NotebookLM to Summarize Videos

1. **Add video to NotebookLM**: Create a notebook with the YouTube video URL
2. **Add to library**: Save the notebook link with descriptive tags
3. **Research**: Ask NotebookLM questions about the video content
4. **Summarize**: Create a markdown document with key points, timestamps, and citations
5. **Save to docs**: Store summaries in `docs/video-summaries/` directory

### Example Commands

- **Summarize video**: `"Summarize https://www.youtube.com/watch?v=n-9bdREECTQ and save to docs/video-summaries/tws-api-essential-components.md"`
- **Research topic**: `"Research TWS API architecture in NotebookLM"`
- **Create documentation**: `"Create documentation from the TWS API video in NotebookLM"`

## Best Practices

### 1. Tag Videos Properly

Use descriptive tags when adding videos to NotebookLM:

- `youtube, tws-api, tutorial`
- `youtube, options-trading, strategy`
- `youtube, ib-api, implementation`

### 2. Organize by Topic

Create separate notebooks for different topics:

- **TWS API Videos**: All TWS API related videos
- **Trading Strategies**: Options trading and box spread videos
- **Implementation Guides**: Code examples and tutorials

### 3. Include Timestamps

When summarizing videos, include timestamps for key points:

- `[12:34]`: Explanation of EClient vs EWrapper
- `[23:45]`: Code example for order placement

### 4. Cross-Reference Documentation

Link video summaries to related documentation:

- Reference `docs/ECLIENT_EWRAPPER_ARCHITECTURE.md` in TWS API video summaries
- Link to `docs/TWS_API_BEST_PRACTICES.md` for best practices

### 5. Update Documentation Index

After summarizing videos, update:

- `docs/API_DOCUMENTATION_INDEX.md` and `docs/RESEARCH_INDEX.md` - Add video summaries
- `docs/YOUTUBE_VIDEOS.md` - Update video status

## Requesting Video Summaries

To request a video summary, use this format:

```
"Summarize [youtube-url] and create a summary in docs/video-summaries/[filename].md"
```

The AI will:

1. Add the video to NotebookLM (if not already added)
2. Research the video content
3. Create a comprehensive summary with key points
4. Include timestamps and citations
5. Save to the specified location
